# 📊 Ejercicio Práctico - Consultas SQL (DB_JS45-7)

¡Bienvenidos! Este repositorio contiene el material y los datos de prueba necesarios para la actividad práctica de consultas SQL. 

En este ejercicio pondremos en práctica el uso de filtros por igualdad, rangos numéricos y patrones de búsqueda de texto mediante comandos como `WHERE`, `LIKE`, `ILIKE` y operadores lógicos.

## 📁 Contenido del Repositorio

* `inserts.sql`: Archivo principal que contiene la creación de la tabla y la inserción de **1000 registros ficticios** para realizar las pruebas.
