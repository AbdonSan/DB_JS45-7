¿Que es Responsive Design?

Responsive Design (diseño responsivo) es una tecnica que permite que una pagina web se vea bien en cualquier dispositivo: celular, tablet, laptop o monitor grande.

El problema con usar valores fijos como width: 800px es que en una pantalla de celular (que mide ~375px) el contenido se desborda. La solucion es usar unidades relativas y media queries.

Unidades Relativas

Valores que se adaptan al contexto: %, em, rem, vw, vh

Media Queries

Reglas CSS que aplican estilos SOLO cuando se cumple una condicion (ej: ancho de pantalla)

Mobile First

Estrategia de diseñar primero para celular y luego agregar estilos para pantallas mas grandes

1. Valores Fijos vs Responsivos

Compara estas dos cajas. Redimensiona la ventana del navegador para ver la diferencia:

Width: px vs %

FIJO
width: 600px No cambia. En pantallas chicas se desborda.
RESPONSIVO
width: 80% Siempre ocupa el 80% del contenedor padre.

max-width: la combinacion ideal

FIJO
width: 500px Siempre mide 500px, no importa la pantalla.
RESPONSIVO
width: 100% + max-width: 500px Crece hasta 500px, pero se encoge si la pantalla es chica.

Guia rapida de unidades

Unidad Tipo Relativa a... Ejemplo Uso comun
px Fija Nada (absoluta) width: 300px Bordes, sombras, detalles pequenos
% Relativa Elemento padre width: 50% Anchos de contenedores, columnas
em Relativa font-size del padre padding: 1.5em Spacing proporcional al texto
rem Relativa font-size del <html> (16px default) font-size: 1.2rem Tipografia, spacing consistente
vw Relativa 1% del ancho de la ventana font-size: 5vw Titulos que escalan con la pantalla
vh Relativa 1% del alto de la ventana height: 100vh Secciones de pantalla completa

2. Unidades de Viewport (vw, vh)

vw y vh son relativas al tamaño de la ventana del navegador (viewport). 1vw = 1% del ancho de la ventana. 1vh = 1% del alto de la ventana.

width: 50vw Siempre mide la mitad de la ventana
width: 25vw Siempre mide un cuarto de la ventana
width: 75vw Siempre mide tres cuartos de la ventana

Texto que escala con la ventana

Este texto tiene font-size: 3vw (escala con la ventana)

Este texto usa clamp(14px, 2.5vw, 24px) (tiene minimo y maximo)

3. Media Queries

Las Media Queries permiten aplicar estilos CSS solo cuando se cumple una condicion. La mas comun es el ancho de la pantalla:

Sintaxis basica:
/* Estilos que SOLO se aplican en pantallas de 768px o menos */
@media (max-width: 768px) {
    .menu {
        flex-direction: column;
    }
}

/* Estilos que SOLO se aplican en pantallas de 769px o mas */
@media (min-width: 769px) {
    .menu {
        flex-direction: row;
    }
}

Breakpoints comunes

Los "breakpoints" son los puntos donde el diseño cambia. Estos son los mas usados:

📱 Mobile 0 - 480px
📱 Mobile Grande 481 - 768px
📋 Tablet 769 - 1024px
💻 Desktop 1025 - 1200px
🖥️ Wide 1201px+

min-width vs max-width

max-width (Desktop First)

Empiezas con estilos de escritorio y vas "bajando":

/* Base: estilos desktop */
.columnas { display: flex; }

@media (max-width: 768px) {
    /* Tablet y menor */
    .columnas { flex-wrap: wrap; }
}

@media (max-width: 480px) {
    /* Mobile */
    .columnas { flex-direction: column; }
}

min-width (Mobile First) ⭐

Empiezas con estilos de celular y vas "subiendo":

/* Base: estilos mobile */
.columnas { flex-direction: column; }

@media (min-width: 481px) {
    /* Tablet y mayor */
    .columnas { flex-wrap: wrap; }
}

@media (min-width: 769px) {
    /* Desktop */
    .columnas { display: flex; }
}

.demo-nav-links {
    display: flex;
    flex-direction: column;
    gap: 5px;
}

/* Desktop: links en fila */
@media (min-width: 769px) {
    .demo-nav {
        display: flex;
        justify-content: space-between;
        align-items: center;
    }

6. Imagenes Responsivas

Las imagenes deben adaptarse al contenedor sin desbordarse. Compara estos dos enfoques:

FIJO
width: 800px; /* Se desborda en pantallas chicas */
RESPONSIVO
max-width: 100%; height: auto; /* Se adapta siempre */
Regla de oro para imagenes responsivas:
img {
    max-width: 100%;   /* Nunca mas ancha que su contenedor */
    height: auto;      /* Mantiene la proporcion */
    display: block;    /* Elimina espacio extra debajo */
}

7. Tipografia Responsiva

Los tamaños de texto tambien deben adaptarse. Aqui hay 3 tecnicas, de la mas basica a la mas moderna:

Tecnica 1: px + Media Queries

Este texto cambia de tamaño con media queries. Es la tecnica mas basica pero requiere definir cada breakpoint. 

p { font-size: 14px; }

@media (min-width: 768px) {
    p { font-size: 16px; }
}
@media (min-width: 1024px) {
    p { font-size: 18px; }
}

Tecnica 2: vw (viewport width)

Este texto escala proporcionalmente con la ventana. Cuidado: puede quedar MUY chico o MUY grande. 

p { font-size: 2vw; }
/* Problema: en 320px = 6.4px (muy chico!)
   En 1920px = 38.4px (muy grande!) */

Tecnica 3: clamp() (La mejor) ⭐

Este texto escala fluido pero con un minimo y un maximo. Es la tecnica mas moderna y recomendada. 

p { font-size: clamp(14px, 2.5vw, 22px); }
/*                   ↑       ↑       ↑
              minimo  |  ideal  |  maximo
              nunca menos   escala    nunca mas
              de 14px     con ventana  de 22px  */
